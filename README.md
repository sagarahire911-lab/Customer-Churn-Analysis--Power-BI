# Customer-Churn-Analysis-Dashboard
📊 Power BI dashboard analyzing customer churn patterns for a telecom company | 27% churn rate | 7K customers | Interactive filters &amp; KPI cards

## 🔍 Project Overview
Analyzed customer churn data of a telecom company using Power BI 
to identify why customers are leaving and what can be done to retain them.

## 📌 Key Metrics
| Metric | Value |
|--------|-------|
| Total Customers | 7K |
| Total Revenue | 16.06M |
| Churn Customers | 2K |
| Churn Rate | 27% |
| Retained Customers | 5K |

## 💡 Key Findings
- Month-to-month contract users churn the most (3.9K)
- Fiber optic internet users have highest churn risk (3.1K)
- Electronic check payment users churn more than auto-pay users
- Customers without Online Security churn 49.6%
- High-value customers leaving = 0.32M revenue at risk

## ✅ Recommendations
- Offer discounts on long-term contracts
- Improve fiber optic service quality
- Promote auto-payment with cashback offers
- Give free trial of Online Security service
